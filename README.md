雪螢個人使用在許个吳語傳統漢字表。

該个傳統漢字標準基於中華人民共和國古籍表漢字標準（大陸繁躰）GB/Z 40637-2021，竝根據古代尤其是宋明以降吳越地區民間常用个俗躰字大規模介字躰搭字形修改脱哉。

箇个傳統漢字標準針對吳越地區語言（吳語/吳方言）个用字習慣專門優化做脱哉。可以講，箇个方案是好專門書寫吳語个繁躰方案。

請注意：箇个方案有意望民間个俗躰常用字形靠攏，也就是擇字上向有「從俗弗從古」个傾向，同辰對於同一字存在多種弗同俗躰字形个情况，雪螢儘量最常用个一躰揀取哉。力微眼拙，箇叻僅供參考。

同辰作為雪螢自用个方案，箇个方案个揀字雪螢个「私貨」搭個人偏好包括哉，箇叻僅供參考。

箇个方案發於公開平臺是為仔存檔以供雪螢自用。還請諒解。
